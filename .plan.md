# Implementation Plan: Better Than VieShow Admin

## 1. Project Overview

Develop an admin dashboard for cinema management, featuring movie management, theater seating layout builder, and screening schedule timeline.

## 2. Current State Analysis

- **Tech Stack**: React, TypeScript, Vite, Tailwind CSS.
- **Key Components**:
  - `SeatingChart`: Visual editor for theater layouts.
  - `Timeline`: Schedule visualization.
  - `MovieForm`: Detailed movie entry form.
  - `Sidebar`: Navigation structure.

## 3. Implementation Roadmap

### Phase 1: Theater Builder Refinement

- [ ] **Seating Chart Interaction**: Improve drag-to-select efficiency and interaction logic.
- [ ] **State Persistence**: Ensure layout saves correctly to backend/local storage.
- [ ] **Validation**: Prevent invalid layouts (e.g., inaccessible seats).

### Phase 2: Movie Management Integration

- [ ] **API Connection**: Connect `MovieForm` to actual backend endpoints.
- [ ] **Image Upload**: Verify Cloudinary or local upload for posters.
- [ ] **Form Validation**: Robust validation using `react-hook-form`.

### Phase 3: Timeline & Scheduling

- [ ] **Drag & Drop**: Implement drag-and-drop for scheduling movies into theaters.
- [ ] **Conflict Detection**: Prevent overlapping schedules.
- [ ] **Calendar View**: Enhanced view for daily/weekly schedules.

### Phase 4: Authentication & Polish

- [ ] **Admin Login**: Secure login flow with token management.
- [ ] **Error Handling**: Global error boundary and toast notifications.
- [ ] **UI/UX**: Consistent styling and user feedback.
